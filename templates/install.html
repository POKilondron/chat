Install Pock Empire

Follow these instructions to run Pock Empire on your computer.

1 Install Python

First, make sure you have Python 3.8 or higher installed.

Download Python

During installation, remember to check the "Add Python to PATH" option.

2 Download the Application

Download the Pock Empire application package:

Download Pock Empire

Extract the ZIP file to a folder on your computer.

3 Install Dependencies

Open a terminal or command prompt, navigate to the folder where you extracted the files, and run:

pip install -r requirements.txt

4 Run the Application

After installing dependencies, start the application with:

python app.py

5 Access the Application

Open your web browser and navigate to:

http://localhost:5000

Or use our quick-launch method by opening:

http://localhost:5000/start

Security Notice

Administrative access is restricted to authorized personnel only.

If you are the administrator, use your personal credentials to access the system.
